Parking Lot Aggregator (FASTag based)

This project is a parking lot aggregator platform just like Zomato, designed to make parking lot booking convenient for customers and management easy for lot owners. It integrates with IoT (FASTag Solution) for automated parking management and real-time updates.

Tech Stack

  • Lot Owner Site: Uses the MERN stack for creating a user-friendly interface for parking lot owners, allowing them to manage their parking spaces effectively.
  • Redis: Implemented for caching, significantly reducing response times and improving data communication between the server and client by up to 80%.
  • Socket.io and Redis: Made a realtime scalable dashboard using redis and socket to sjow the status of parking lot effectively.
  • IoT Module: The IoT integration uses ESP32 to manage RFID (FASTag)-based entry and exit, providing automated and secure access for vehicles along with billing.
  • Client Site: Built with the MERN stack (MongoDB, Express, React, and Node.js), which provides a robust, scalable framework for managing customer interactions and bookings.
  • Mapbox API: Integrated to display dynamic, interactive maps of parking lots location based on lot pincode.
  • Stripe: This project uses Stripe as the payment gateway, ensuring secure and seamless online transactions for parking bookings.
  • Nodemailer: Integrated for sending automated email notifications to users about their booking confirmations and reminders.

Deployment

  • Deployed on AWS: The entire platform is hosted on AWS, ensuring high availability, scalability, and security for handling multiple users and large datasets.
  • Nginx: Used as a reverse proxy and load balancer for efficient and smooth deployment across servers, ensuring that traffic is evenly distributed and managed.

Features

  • Customer and Owner Sites: The platform provides two different sites â€” one for customers to find and book parking spots, and another for owners to manage their parking lots, ensuring flexibility and ease of use for both user types.
  • Multiple Booking Methods: Customers can choose from three types of bookings:
    • Hourly bookings: Book parking spaces for short durations.
    • Date-wise bookings: Reserve parking for an entire day or specific date ranges.
    • On-the-spot offline bookings: Book a parking space without pre-reserving, useful for walk-in customers.
  • RFID(FASTag)-Based Automated Entry and Exit: The IoT module with RFID(FASTag) sensors automates vehicle entry and exit from parking lots, reducing the need for manual checks and increasing efficiency.
  • Optimized Parking Allocation: The system includes algorithms that help manage potential booking conflicts and optimize the use of parking spaces. For instance, the same parking space can accommodate multiple hour-wise bookings on the same day to maximize utilization.
  • Flexible Space Management: The platform supports two types of parking spaces â€” one reserved for online bookings and the other for offline, on-the-spot bookings. These spaces can be interchanged based on demand, ensuring maximum utilization of available parking lots.
  • Real-Time Dashboard: Owners can view and manage parking lot status in real time using Redis and Socket.io, which ensures that data is constantly updated and synchronized across the platform.
  • Email Notifications: Customers receive automatic email notifications about their bookings.
  • Interactive Map Integration: Mapbox is used to give customers a visual representation of nearby location of parking lots, enhancing the user experience by providing clear and interactive maps.
  • Payment Integration with Stripe: The use of Stripe ensures secure, seamless, and quick transactions for customers making online bookings.
