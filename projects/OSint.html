OSint SaaS

This OSINT project offers Software as a Service (SaaS) for open-source intelligence gathering. It includes phone OSINT, email OSINT, social media OSINT, and reverse image search capabilities. Designed to streamline intelligence workflows, it provides comprehensive, accessible tools for investigative needs.

Tech Stack

  • Frontend: Developed using React and JavaScript for a responsive and user-friendly interface.
  • Backend: Built with MongoDB, Node.js, and Express to handle data management and server operations.
  • Multer: Used to store images on the backend for the reverse image search functionality, ensuring smooth image uploads and processing.
  • Payment Integration: Uses Stripe as the payment aggregator to process secure transactions within the app.
  • Instagram API: Integrated for accessing social media data as part of the OSINT features.
  • NumVerify API: Used to validate phone numbers, ensuring accurate and verified user data.
  • Email Reverse Lookup API: Implemented to perform email verifications and enhance the OSINT services.
  • SERP API: Utilized for reverse image search, allowing efficient OSINT image-based intelligence gathering.

Deployment

  • Deployed on AWS: The entire platform is hosted on AWS, ensuring high availability, scalability, and security for handling multiple users and large datasets.
  • Nginx: Used as a reverse proxy and load balancer for efficient and smooth deployment across servers, ensuring that traffic is evenly distributed and managed.

Features

  • SaaS Model: Operates on a pay-per-search basis, allowing users to pay as they go for each search performed.
  • User Account Management:
    • Provides login, signup, and registration functionalities for easy account access.
    • Includes a balance management system that lets users add funds via Stripe when their balance is low.
  • OSINT Services:
    • Phone OSINT: Validates phone numbers and provides details like location, carrier, country, and line type.
    • Email OSINT:
      • Checks email validity and disposability.
      • Offers domain-related information such as MX, IP, and MX records.
      • Detects email forwarders and confirms if an email is whitelisted.
    • Social Media OSINT: Retrieves Instagram and Twitter profile data using just the username.
    • Reverse Image Search:
      • Uses Google Reverse Image Search for image-based intelligence gathering.
      • Employs Multer to upload and store images for analysis.
